# Contact manager application

Name:        contact-manager-app

Description: An application to manage list contact over many company

# The technologies used

- JDK 11
- Spring boot
- JPA
- Lombok
- H2 Memory Database
- JUnit
- Mockito
- Maven
- Spring security with Jwt
